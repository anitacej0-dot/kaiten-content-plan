# inputs/current_cycle — данные текущего цикла

Сюда складываются входные данные месяца. **Реальные файлы не попадают в GitHub**
(защищены `.gitignore`), в репозитории — только этот README и шаблоны в
`inputs/templates/`.

## Что положить (минимум — только контент-план)

| Файл | Обязателен | Что это |
|---|---|---|
| `*.xlsx` контент-план | да | предыдущий/текущий план (как есть, из Kaiten-редакции) |
| `analytics/` (папка) | нет | сюда — выгрузки Roistat (`Report`) и Метрики (`Отчет`) |
| `seo_signals.csv` | нет | SEO-запросы/кластеры (шаблон в `inputs/templates/`) |
| `editorial_ideas.csv` | нет | идеи редакции |
| `mandatory_publications.csv` | нет | обязательные публикации (запуски, анонсы) |
| `product_priorities.md` | нет | приоритетные продуктовые направления |
| `sales_kam_signals.md` | нет | запросы продаж и KAM |
| `cases_and_experts.md` | нет | доступные кейсы и эксперты |
| `editorial_capacity.md` | нет | мощность редакции (материалов/нед, авторы) |
| `launches_and_deadlines.md` | нет | запуски и дедлайны |
| `decisions.csv` | нет | ручные решения главреда (см. ниже) |

Файлы Roistat и Метрики распознаются по содержимому — имена могут быть любыми.

## Как начать

1. Скопируйте нужные шаблоны из `inputs/templates/` сюда и заполните.
2. Контент-план Excel положите прямо в этот каталог (файл не изменяется).
3. Выгрузки аналитики — в `analytics/`.
4. Запустите `/new-plan` (или `python scripts/run_pipeline.py`).

Если какого-то файла нет — pipeline не остановится: он отметит пропуск и снизит
уверенность соответствующих выводов (см. `01_input_audit.md` и `plan_summary.md`).

## Ручные решения (`decisions.csv`)

Колонки: `content_id, field, new_value, author, date, reason, is_exception`.
- `field = decision_status`, `new_value = утверждена|выбрана|в бэклог|отклонена` —
  перемещает тему между разделами плана.
- любой другой `field` (например `editor_comment`, `author`, `target_week`) —
  меняет значение поля инициативы.

Решения — это ВХОД: при каждом запуске они применяются заново, поэтому повторный
прогон их не затирает.
