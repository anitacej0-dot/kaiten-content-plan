# Мощность редакции на цикл

Pipeline читает отсюда число материалов в неделю и число авторов (по ключевым
фразам). Формулируйте просто, например:

- Редакция выпускает **5 материалов в неделю**.
- Доступно **3 автора** (+ аутсорс на подборки).
- Не более **2 тяжёлых материалов** (кейсы/исследования) в неделю.

Если файла нет — берутся значения по умолчанию из `config/portfolio.yaml`.
