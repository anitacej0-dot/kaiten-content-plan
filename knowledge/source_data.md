# Источники данных для планирования

Pipeline принимает не один список SEO-тем, а набор независимых сигналов.

## Обязательные

- предыдущий контент-план и история публикаций;
- актуальные продуктовые приоритеты и запуски;
- SEO: спрос, позиции, страницы, кластеры, каннибализация;
- аналитика блога: посещаемость, вовлечение, маршруты;
- доступные лидовые и коммерческие действия;
- данные продаж/KAM: вопросы, возражения, сегменты, сценарии;
- список кейсов и доступных экспертов;
- мощности редакции на цикл;
- ограничения и требования площадок.

## Дополнительные

- конкурентные пробелы;
- PR-поводы;
- сезонность;
- темы, которые уже обещаны партнёрам;
- темы, которые нельзя брать в этом цикле.
